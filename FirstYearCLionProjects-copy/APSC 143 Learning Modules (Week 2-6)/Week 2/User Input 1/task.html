User Input

Another important concept to programming is taking input from the user using the scanf function, which allows code to run using different values each time. In this example, the program allows the user to input an integer value, which is stored in the variable userInt and output with the printf function.

Make note of the key components of the scanf function: the format specifiers and the variables preceded with the '&' character. Try running the code and inputting different values to see how it works. You can input a value in the Run tab at the bottom of the screen by typing it into the terminal and pressing the Enter key.
