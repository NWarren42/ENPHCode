Math Library

Rounding Functions

The math library also has built-in functions for rounding float values to an integer: the floor function, the ceiling function, and the round function.

In this task, add one of these functions to the program so that the output will always round up from the original float value. If you need a hint, check below!

floor(x) always rounds down. ceil(x) always rounds up. round(x) follows usual rounding rules.
