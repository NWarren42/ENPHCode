Variable Incrementation

Incrementation is incredibly useful in many programming scenarios, such as keeping count of something or making iterations of a section of code (more on that in week 4). For now, let's examine the differences between pre-increment and post-increment using an int variable x.

Try running the program and see what the resulting values for x are. Does it make sense? The postincrement x changes only after the output statement, while the preincrement x will change before the output statement is executed.

Try switching the order of the two printf functions to see how the resulting values changes. Feel free to experiment with different combinations of incrementation, or even deincrementation (x-- or --x).
