Math Library

Mathematical Functions

Now that we can use expressions effectively, we can use the math.h library to access any number of mathematical computational functions to assist us in our programming.

For this example, we'll use two of the more commonly used functions: the exponent function and the sine function. Using the defined float variable x, compute the cube and the sine of x. The following values should appear:

The cube of x is 199.18

The sine of x is -0.43

If you need help with the functions, see the hints below. Remember that you need to include the math library to access these functions!

Exponent function: pow(base, power)
Sine function: sin(value)
