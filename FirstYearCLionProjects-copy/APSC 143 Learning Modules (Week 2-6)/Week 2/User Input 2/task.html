Multi-Variable User Input

The scanf function can also be used to take multiple values simultaneously by including multiple format specifiers inside the double quotes. The format inside the quotes should match the format of the input.

Examples:

    56 4.7 -> is formatted as -> "%d %f"
    p, 23.5 -> is formatted as -> "%c, %f"

Try updating the scanf statement from the previous example to read an int followed by a char, separated by spaces. You will have to make a new variable to hold the char value. Then, print them both in the following format:

int = 35, char = 'u'

In order to pass the output test, you will need to input the integer 35 and the character 'u' separated by spaces into the Run tab below once you have started the Check operation. You are encouraged to run your code using different inputs to test the limitations of your program.

As always, print a new line character to include an empty line at the end of your output. For assistance with the scanf input, see the below hint.

scanf("%d %c", &userInt, &userChar);
