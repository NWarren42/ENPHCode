Char Type Variables

For the next step, we want to change the example code to output a char type variable, which is any single character, instead of an int type. You will have to change the type and value of x, as well as the format specifier in the printf function.

Change the code to output the following:

x = q

The value for a char type is initialized using single quotes around the desired character. For example, '@'.
The format specifier for a char type is %c.
