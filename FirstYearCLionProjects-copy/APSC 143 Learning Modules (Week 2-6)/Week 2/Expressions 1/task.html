Product of Two Values

One of the most useful features of computer programming is the ability to program mathematical expressions. The next few tasks will have you work through some simpler operations.

For this task, declare a float value z that is initialized as the product of x and y. If done correctly, you will get the following output:

The product of 2.48 and 6.07 is 15.05

Keep in mind that we could just as easily use a scanf function to ask the user for the two factors, to make the program more robust. This will be useful in the coming assignments.
