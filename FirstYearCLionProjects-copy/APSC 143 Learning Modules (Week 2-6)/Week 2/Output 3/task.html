Escape Character

The new line character is a special use of the escape character, which is represented by the backward slash character \ . The escape character is usually used to print characters that typically have other purposes.

For example, since '\' is used as the escape character it cannot be output in the usual way. This issue can be avoided if we output it using the following format:

"\\"

For this task, write a prinf statement to output the following sentence:

"Back to the Future" is a great movie.

Remember that you will need to use the escape character to print the double quotation marks. You will, as always, also need to include a new line character at the end of the statement. See the hint below if you get stuck.
