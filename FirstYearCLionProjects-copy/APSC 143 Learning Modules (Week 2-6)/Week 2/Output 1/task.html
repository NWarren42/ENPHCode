Output Statements

Next, we'll write our own printf statements. There are two comments inside the main loop that we want to replace. Try to write a pair of statements that output:

Welcome to APSC 143.

Computer programming is fun!

Make sure you add an include statement, and also print a new line character at the end of each! If you are stuck, take a look at the hint below.

printf("text here!\n");

This task, along with most following tasks, will test your code against a sample solution to see if you got it right. Run the program using the Check button below to see if your code is correct. If not, you can use the peek solution option to compare your code with the sample solution, or the compare output option to see how your output differs from the expected output.

Keep in mind that when you use the Check button, any output that occurs will be displayed in the Run tab of the bottom panel, which is called a terminal window.

If you want to run your code without comparing it with the sample solution, you can still use the green play button that appears on the left hand side of your code. This is a useful tool for experimenting with your code to see what its limitations are.
