Prompted User Input

Each coding assignment in this course will ask you to include a prompt for any user input; meaning the program must print a statement asking for a desired input. This task will help teach you how to design these prompts, identically to how it will be used in assignments.

Essentially, a prompt will be a print statement that specifies the requested variables that will be read in the following scanf statement. In each assignment, the prompt does not print a new line character, allowing the values to be input on the same line. Variables will always be separated by spaces.

In this example, the program requires the height of a building in meters, and the number of floors it contains. Write a prompt that outputs this line:

Enter building height and number of floors:

This prompt should also print a single space after the colon.

When you run the code and enter 312.8 and 26 separated by spaces, you should get the following output and pass the output test:

This building is 312.80m tall and has 26 floors.

Once you have this code correct, you can use it in all of your assignments provided you update it for the necessary values.
