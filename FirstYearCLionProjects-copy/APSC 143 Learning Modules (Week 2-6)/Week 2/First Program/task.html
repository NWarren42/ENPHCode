First Program - Hello World

Welcome to the APSC 143 interactive learning module! The module is designed to give you step-by-step instructions on building simple programs and learning the key concepts from the lecture videos. While these tasks are optional, they are highly recommended as programming is best learned through practice.

As you can see on the left hand directory, there are multiple tasks for each week of the module. When you successfully complete a task, The checkbox will turn green.

To begin, try running this "Hello World" program by using the green play button on the left side of the code, clicking the first option that appears. If all goes well, you will see Hello, World! appear in the terminal window at the bottom of the screen.

Clicking the Check below this text will mark down that you have completed this task. However, most upcoming tasks will evaluate the code you write to make sure you understand the concepts.

Understanding the Code

This program uses a few concepts that are critical to programming in C. First, we use the main function to tell the compiler what code we want executed, in the order we want it. Second, we have an include statement, which tells the compiler that we are using functions that are found in the stdio.h library, such as printf. Keep in mind that the return 0 statement at the end of the program is optional for the main function.
