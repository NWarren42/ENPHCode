New Line Character

What does the new line do?

Why do we use the new line character on all of our output statements? Here's a chance for you to experiment and find out for yourself. Copy your code from the main.cpp file in Week 2 -> Output 1 and try removing the new line character from your first output statement. Run your code and see what happens!
