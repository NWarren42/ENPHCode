Float Type Variables

This next task will help you use float type variables in programming and in output statements using format specifiers.

Fill in the missing code so that it outputs the following, without changing the initialized value:

x = 6.845

Remember to print a new line character, as always.

Float type variables use the format specifier %f. The number of decimal places can be specified by including ".[n]" between the '%' and the 'f', where n is the number of desired decimals.
