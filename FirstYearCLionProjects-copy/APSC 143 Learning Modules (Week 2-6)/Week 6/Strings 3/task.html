Strings

Like any array, strings can have their elements individually reassigned. This task provides a fun example through user input.

This example uses an "infinite" loop that will modify the string boggle based on the index and new letter input by the user. Write code to perform the following:

    Declare the string boggle and initialize it to hold the word "banked".
    Check if the value for pos is a valid index: If true, replace the letter at the chosen index with the new letter, key.

To complete the task, input the following combinations in tis order:

3 d 0 r 2 d 1 i 5 n 8 y

If you need help, take a look at the hints below. Feel free to experiment with different entries and see what you get!

The index needs to be between 0 and the length of the word.
