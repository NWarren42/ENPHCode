Arrays of Strings

When using strings, a 2-dimensional array essentially becomes a 1-dimensional array of strings - since one of the dimensions holds the letters making up the strings. This task will dive into how we use these more comprehensibly.

The example code uses a string array called list. Declare this array, and store the following 6 strings in this order:

giraffe elegance osmosis universal victorian romania

The rest of the program iterates through the array, printing each string and the string in reverse. The strings are printed in order. The forward and reverse versions are separated by a single space. You will need to write a statement that will print each char in the string in reverse order.

Don't forget about our new favourite function strlen! If you get stuck, check out the hints below.

Example string array: char names[2][10] = {"Angela", "Muhammad"};
You can indicate the current char like in a normal 2D array: list[word index][letter index]

As you will see in the next few tasks, string arrays can be just as useful as any other array! We'll investigate other ways to create string arrays in later weeks.
