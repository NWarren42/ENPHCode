String Initialization

Welcome to week 6! We'll be going over strings and their many uses. A string is a 1D array of chars that has some special properties which set it apart from other arrays. The next tasks will investigate this!

Strings can be initialized on one line, like other arrays; however, they require an additional null character, \0, at the end of the array. Initialize the string word with each individual character in "pear".

The program has also defined the variable average. Write some code that will find the average value of each letter (excluding the null character) in the array. The variable wordsize has been created so that you can easily find the length of the word (excluding the null character).

Lastly, you will need to output word, which can be done using a single printf statement. Use the below format for the printf statement. Check the hints if you need any help.

Word = "pear"

char identifier[size] = {'a', 'b', ... '\0'};
Your for loop should run from 0 to wordsize - 1, inclusive.
