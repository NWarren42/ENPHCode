The strcpy Function

This function can be used to copy one string into the memory of another - which has a similar use to memcpy.

This task is centred around the array of strings called list, which holds 6 names on it. Using a while loop, the user is prompted to enter two indexes to swap in the list (e.g. swapping 0 and 1 would put "Bill" before "Charlie"). If they enter invalid indexes or the same index twice, the loop ends and the final list is printed.

Your task is to use the strcpy function to swap the names of the two indexes entered by the user. The string temp has been declared to help with this. Be careful about overwriting the names! If you need help, take a look at the hint below.

Copy the x name to temp -> copy the y name to x -> copy the temp name to y

To complete the task, make 2 swaps so that the list is in alphabetical order before entering an invalid input.
