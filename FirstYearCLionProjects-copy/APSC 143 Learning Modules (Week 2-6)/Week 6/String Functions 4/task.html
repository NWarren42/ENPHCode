The strcat Function

This function allows us to combine 2 strings together by adding one string onto the end of the other. Let's explore its uses!

The array of strings called options is a collection of strings to be stitched together by the program. The string sentence will hold all the chosen words. Your task is to write code within the while loop that will:

    Read an integer into the variable index, which represents which word should next be added to sentence;
    Check if the index is invalid (if invalid, break the loop);
    Using the strcat function, add the corresponding word in options to the sentence string.
