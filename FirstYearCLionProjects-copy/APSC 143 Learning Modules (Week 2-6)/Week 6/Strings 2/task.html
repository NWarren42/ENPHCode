String Initialization

There is a final way we can initialize strings, which is significantly easier than the others: omit the array length and use " " around the string.

This example finds the largest and smallest letters found in the string word. Initialize word to hold "watermelon". Then, declare and initialize two char variables min and max to hold the largest and smallest letters.

You will need to create a loop to check each element in word and update min and max by comparing to each element. If you get stuck, take a look at the hints below.

Set the initial value of min and max to the first character in word.
The loop should run from index 0 to the length of the word - 1. Inside, check if the current element is smaller than min, and if it's larger than max.

While each initialization method is useful in different situations, choose the method you like the best and stick with it!
