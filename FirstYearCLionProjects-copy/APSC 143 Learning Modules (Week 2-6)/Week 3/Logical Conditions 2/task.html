Logical Conditions

As seen in the previous task, logical conditions can be assigned to int type variables to be evaluated. Here, we have another, simpler version.

To complete this task, you need to create a logical expression that will evaluate true when y is larger than double the value of x. If you get stuck, check the hint below.

y > (x doubled)

Keep in mind how simply we could change the expression to have y greater or equal to the doubled value of x, or even have this condition evaluate to false (using the ! operator). You are encouraged to try different values for x and y to see how the condition evaluates in different situations.
