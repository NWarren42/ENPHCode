Logical Conditions

The OR Operator

The OR operator, written as ||, is similar to the AND operator with a key difference: it evaluates true if at least one of the logical expressions evaluates to true. Therefore, it will evaluate to true no matter how many false conditions are present, so long as there is at least one condition that is true.

Using the example code, write a new logical expression that will evaluate to true if any of the following conditions are true:

    The value for y is less than or equal to the value for x.
    The value for y is a negative number.
    The value for x is a negative number.

For this task, 0 is a positive value. If you get stuck, take a look at the hint below.

y is less than or equal to x || y is less than 0 || x is less than 0

As always, you are encouraged to experiment with the conditions and values to see what changes. Furthermore, think about how the AND operator and the OR operator could be used simultaneously!
