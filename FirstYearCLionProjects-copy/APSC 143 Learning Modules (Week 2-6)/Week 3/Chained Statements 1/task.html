Chained If-Else Statements

We can check for multiple conditions individually using a chained if-else block. Each statement of the block is treated as its own if statement and run if true; however, only the code contained in the first true statement will run. The rest of the statements will be skipped. Let's code one for ourselves to see how it works!

The example code uses a basic if-else block that you will add statements to. Between the if and the else blocks, write an else if statement with the condition that the difference between x and y is 5 or less. Remember that y must be at least as large as x if this condition is being checked! Inside this block, output the following statement:

y is a little larger than x.

Then, add another else if statement after the one you just wrote. This one has the condition that the difference between x and y is 50 or less. Inside this block, output the following statement:

y is significantly larger than x.

If you get stuck, check the hints below. Do you know which printf function will execute?

else if(difference <= 5) { output 1 }
else if(difference <= 50) { output 2 }

Consider the order in which we wrote the else if conditions. Since the third condition contains the second (ie. any number that is less than 6 is also less than 51), we have to check for the more specific condition first. Otherwise, the larger condition would evaluate true before the smaller condition could even be checked! Try reordering the conditions to see for yourself.

Feel free to check different values for x and y, and keep in mind that we can have any number of statements in the chain. Also, the else at the end is optional and used when you want a catch-all for any conditions that fail everything else.
