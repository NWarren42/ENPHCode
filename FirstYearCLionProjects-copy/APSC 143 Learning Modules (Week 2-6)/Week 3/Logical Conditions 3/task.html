Logical Conditions

The AND Operator

The AND operator, written in code as &&, is useful in situations where we want all our required conditions to evaluate as true. This task will display how a condition using this operator works.

Using the code from the last task, add to the condition for the variable z such that both values for x and y must be positive (for this example, 0 is a positive number) in addition to the previous condition - that y is greater than the doubled value of x.

Remember that you can use as many AND operators in a condition as necessary! If you need help, see the hint below.

(previous condition) && x must be greater than -1 && y must be greater than -1

As always, you are encouraged to enter different values for x and y to see how the condition evaluates differently.
