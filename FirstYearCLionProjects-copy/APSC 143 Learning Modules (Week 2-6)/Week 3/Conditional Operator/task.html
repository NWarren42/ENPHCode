Conditional Operator

The conditional operator, ?, can be used in place of an if-else block for single-code-line options, or to assign values or expressions to variables based on a logical condition.

For this task, code has been provided for the user to enter any integer. Write a conditional operator statement that will determine whether the entered integer is even or odd. Then, do one of the following:

    If even, print: x is even
    If odd, print: x is odd

To complete the task, enter an odd integer when prompted (using the Run tab).

See the hint below if you get stuck. As seen in the lectures, you can also use this operator to assign one of two values or expressions to a variable. However, this is only an alternative to an if-else statement and so is optional.

(remainder of x / 2 is zero) ? even print command : odd print command;
