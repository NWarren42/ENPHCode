2-Dimensional Arrays

Copying Arrays

If we want to add additional elements to a full array, the only way to do so is to create a new, larger array and copy all of the elements over. This task will go over the two ways to perform this.

The original array, wave1, holds time values in its first row and positions of the sine wave in its second row (based on the first row times). All contents of wave1 have been copied into wave2 using a for loop, which also has a third row containing the cosine wave along the same domain.

For this task, create a third array called wave3 that will store all the data from wave2 - copy this using memcpy. This array will also have a fourth row, to hold the tangent wave along the same domain. When completed, your output will look like this:

x sin cos tan
0.000 0.000 1.000 0.000
1.047 0.866 0.500 1.732

Take note on how the sizeof function is used; the memory of one row is divided by the memory of one element to find the elements in a row.

Copying arrays is a useful tool; choose whichever method you prefer, and master it! (But remember that memcpy can only be used when adding extra rows)

memcpy(bigger array, smaller array, size of smaller array);
