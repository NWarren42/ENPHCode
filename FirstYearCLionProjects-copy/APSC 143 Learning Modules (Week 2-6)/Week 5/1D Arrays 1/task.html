1-Dimensional Arrays

Welcome to week 5! This week's module looks at arrays, an incredibly useful structure for programming.

This example uses an int type array called ray, which holds 5 integers. Each value will be printed using the provided printf function. The values are as follows:

87, 12, 63, 101, 20

To complete this task, declare the array and initialize all of its values on the same line. If you need help, take a look at the hint below.

int identifier[size] = {val1, val2, ...};

This method of array initialization is easy, as long as you know all the values when you declare it.
