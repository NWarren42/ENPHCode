2-Dimensional Arrays

Arrays with 2 or more dimensions can be used in the same ways as their 1-dimensional counterparts, but allow for better organization of their respective variables. Using multidimensional arrays can also be slightly more complex, as we're about to see!

The provided code has declared a 2-dimensional array called nums. Initialize it to hold the following values, using a single line of code:

4 19 10 22
23 7 2 13
14 0 27 8

The code also includes a nested for loop that iterates through each element of the array (make sure you understand how this works). Inside this nested loop, write a printf function to output the current element followed by a tab character, \t. Notice how we have the new line on its own! This will separate each row.

The next task will look into another way to initialize multidimensional arrays, using a nested for loop.

When initializing a 2D array, use { } to separate each row with commas between them.
The current element in the nested loop can be accessed using nums[row][col].
