1-Dimensional Arrays

In this example, we'll use a loop to help us initialize an array. This can allow us to choose individual numbers for each value.

This example uses a float type array nums of size LENGTH(allowing us to modify the array size easily). You will have to declare this array . Each element of nums will be entered separately by the user.

Write a loop that will iterate through each index of nums. On each iteration, get a float value from the user using scanf and store it in the corresponding index of nums. Look at the printf loop for help - yours should end up very similar!

To complete this task, enter the five numbers shown below. If you get stuck, take a look at the hints below.
