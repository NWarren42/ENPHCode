2-Dimensional Arrays

2-dimensional arrays can also be initialized element-by-element using a nested loop, where each loop counter references either the row or column position in the array. This task will explore this idea.

The starter code has defined a 2D array called timesTable of size 12x12, which will hold a multiplication table from 1 to 12. Complete the nested loop code to initialize all elements in the array, so that they follow the following format:

1 2 3 4
2 4 6 8
3 6 9 12
4 8 12 16

Keep in mind that your loop counter variables will be named row and col; you will need to determine which should be in which loop. Also, remember that the array indexes start at 0, but our multiplications should start at 1. The hints below will answer these problems.

Note how the nested loop is written for both initialization and output! We also see the advantage of using #define here: the size of our table can easily be modified with one small change. Try experimenting with resizing the array and different expressions for assigning each element!

Your outer loop should iterate through rows; that way, you can print each element that belongs on the same line as you go.
If both your loops iterate from 0 to 11, the current element should store (row+1) * (col+1).
