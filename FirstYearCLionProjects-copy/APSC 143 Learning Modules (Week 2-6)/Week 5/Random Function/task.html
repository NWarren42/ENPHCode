The Random Function

Now let's use arrays to introduce a new function: rand().

rand() will return an integer value that is between 0 and a very large value. Generally, we want a value in a specific domain, which can be generated with the following expression:

int x = rand() % (max - min + 1) + min;

where max is the largest value of the domain and min is the smallest. The modulus operator will restrict the value to a certain range, while the min addition will translate the range to the desired domain.

For testing purposes, we seed the random function so that it returns the same random values each run. To do this, call srand(int) before any rand() calls are used. Use 2 as the argument.

Both of these functions are part of the standard library. You can include it with the tag stdlib.h.
