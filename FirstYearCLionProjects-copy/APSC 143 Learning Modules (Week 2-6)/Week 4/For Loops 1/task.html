For Loops

The other type of repetition structure, the for loop, is incredibly useful in any number of situations; particularly those that require a set of code to run a specified number of times.

The example code shows a simple for loop used alongside a scanf function. The program asks for a number of loops from the user, then runs the code contained in the for loop that many times. Try entering some different values (using the window of the Run tab) and see how it works.

Notice how the printf function in the loop uses i + 1 as the loop number (since we set i to an initial value of 0). Therefore, i will take on the values 0 to loops - 1 as the loop iterates.
