While Loops

Welcome to week 4! This week's module will be diving into repetition structures and their many uses.

This first example shows a while loop that uses an int value called key to determine how many times the code will loop for. On each iteration, the program takes an integer from the user to add to a total and store in key. Once the user enters 0, the program outputs the total and ends.

Try running the code and entering some values, including negatives! Once you have a good understanding of the code, feel free to experiment with the exit condition or other aspects.
