While Loops

This example will have you write your own while loop.

The provided code declares two int type variables: count, which holds the number of loop iterations, and total, which stores the sum of all the values of count. Write a while loop that does the following:

    On each iteration, add the value of count to total. Then, increment count by 1.
    Exit the loop once total has a value of 100 or greater.

When done correctly, you will get the output: Loop ran 15 times.

As always, you are encouraged to experiment with different exit conditions or iteration components. If you get stuck, take a look at the hint below.

inside the loop: { count++; total += count; }
