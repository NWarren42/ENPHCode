For Loops

This should look familiar! Just like in Do While Loops 1, we will take the same looping situation from While Loops 2 and convert it to a for loop with the same functionality.

As a refresher: this for loop should iterate using count, and exit once total has reached 100 or greater. On each iteration, the value of count will be added to total.

Copy the loop code from either of the two tasks listed above into the placeholder and rewrite it to use a for loop instead. If you need help, take a look at the hint below.

for(count; total is less than 100; increment count) { ... }

Like the previous two, you should get the identical output of: Loop ran 15 times. As shown, for loops can be used as more than just a counter of a certain number of iterations. However, keep in mind that any for loop can be replaced by a while loop, though this can often end up quite complex.
